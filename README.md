# Usage
<!-- usage -->
```sh-session
$ npm install -g react-lib-scripts
$ react-lib-scripts COMMAND
running command...
$ react-lib-scripts (-v|--version|version)
react-lib-scripts/0.0.3 darwin-x64 node-v8.11.2
$ react-lib-scripts --help [COMMAND]
USAGE
  $ react-lib-scripts COMMAND
...
```
